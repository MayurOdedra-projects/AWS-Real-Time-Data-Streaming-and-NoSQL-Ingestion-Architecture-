Objective: Design and deploy a serverless real-time data streaming pipeline to ingest, process, and store high-velocity network 
telemetry data on AWS. 
Tools & Technology: Amazon Kinesis Data Streams, AWS Lambda, Amazon DynamoDB, AWS Management Console. 
Process: 
• Provisioned an Amazon Kinesis Data Stream to capture and buffer continuous network telemetry generated by distributed 
hardware devices.  
• Developed an AWS Lambda function to automatically process incoming stream events and transform data into real time.  
• Configured the Lambda function to write processed records into Amazon DynamoDB, enabling scalable, low-latency 
NoSQL data storage.  
Result/Impact: Delivered a scalable, cost-efficient, and resilient real-time data ingestion pipeline that supported continuous 
streaming workloads and provided a strong foundation for future analytics and network optimization.
